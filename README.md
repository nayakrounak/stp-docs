# Home

Este GitBook funciona como uma base de conhecimento central para **runbooks de deployment** e **documentação de projetos** de várias iniciativas de transformação digital entregues em **São Tomé e Príncipe (STP)** pela **Ooru Digital Private Limited**.

Inclui orientação prática, focada na implementação, tais como:

* **Documentação de deployment** (preparação de ambientes, configuração de infraestrutura, passos de instalação, gates de validação)
* **Referências de arquitetura e topologia** (blocos lógicos, zonas de rede, políticas de exposição)
* **Documentação de operações** (monitorização, logging, backups, DR, runbooks, troubleshooting)
* **Documentação de integrações** (interfaces, controlos de segurança, padrões de troca de dados, reconciliação)
* **Notas e decisões específicas do projeto** (pressupostos, restrições, baselines de configuração, histórico de alterações)

Cada secção de projeto está estruturada para suportar tanto o **deployment inicial** como as **operações day‑2** e destina-se a stakeholders técnicos do Governo, integradores de sistemas e equipas de operações/suporte.

> **Nota:** O conteúdo é mantido e atualizado à medida que as implementações evoluem, devendo ser tratado como a referência oficial para os deployments e procedimentos operacionais documentados.
